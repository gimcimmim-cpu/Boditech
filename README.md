# 🚀 AI 기반 주간회의자료 자동 취합 시스템 (Boditech Med)

![Boditech Med Logo](https://www.boditech.co.kr/assets/images/logo.png) <!-- 로고 URL은 사내 상황에 맞게 수정하세요 -->

본 프로젝트는 **바디텍메드(Boditech Med)**의 사내 업무 효율성을 높이고, 구성원들이 **AI와 Python 스크립트 기반의 업무 자동화(RPA, Robotic Process Automation)**를 이해하고 실무에 도입하기 위해 기획된 기초 프로젝트입니다.

---

## 💡 프로젝트 기획 배경

매주 진행되는 각 부서(팀)별 회의 자료 취합 업무는 반복적이고 소모적인 시간이 많이 발생합니다. 
*   **기존 방식:** 담당자가 Outlook 메일을 일일이 확인 -> 첨부파일 다운로드 -> 미제출 부서 파악 후 독촉 -> PowerPoint를 열고 슬라이드를 수동으로 이어 붙이기 -> 최종 저장 및 공유
*   **도입 후 (AI/RPA):** **단 원클릭(1-Click)으로 메일 필터링, 미제출 부서 체크, 순서에 맞춘 병합까지 약 5초 만에 완료**

이를 통해 취합 담당자의 업무 스트레스를 줄이고, 바디텍메드의 핵심 가치인 '빠르고 정확한 진단'처럼 사내 업무 처리 속도 역시 혁신적으로 단축하는 것을 목표로 합니다.

---

## 🛠️ 주요 기능 (Features)

1. **지능형 이메일 필터링 (Outlook 연동)**
   - 지정된 날짜(오늘)에 수신된 메일 중 조건에 맞는 파일만 1차 추출
   - 파일명 내 '오늘 날짜(예: 0521)' 및 '회의자료' 키워드 동시 검증 (지난 주 자료 오제출 방지)
2. **자동 제출/미제출 현황 리포팅**
   - 사전에 정의된 팀/담당자 리스트와 수신된 메일 발신자를 크로스 체크
   - 즉각적인 콘솔 및 UI 로그를 통해 미제출 부서 신속 파악 가능
3. **PowerPoint 슬라이드 자동 병합**
   - 지정된 서열(Rank)에 맞추어 1순위 부서부터 8순위 부서까지 순차적 슬라이드 삽입
   - 서식 깨짐 없이 원본 디자인을 유지하는 COM 객체 제어 방식 적용
4. **사용자 친화적 GUI (Graphic User Interface)**
   - 파이썬이나 코드를 모르는 일반 임직원도 쉽게 사용할 수 있도록 버튼 클릭 형태의 UI 제공
   - 작업 완료 후 병합된 결과물 폴더 자동 팝업

## ⚙️ 시스템 아키텍처 및 사용 기술

이 프로젝트는 생성형 AI(LLM)의 코드 작성 지원을 받아 기획 및 개발되었으며, 사내 폐쇄망 환경에서도 안전하게 동작하도록 외부 API 통신 없이 Local PC 자원만을 활용합니다.

- **Language:** Python 3.x
- **Libraries:**
  - `pywin32` (`win32com.client`): Microsoft Outlook 및 PowerPoint 애플리케이션 직접 제어 (COM Interface)
  - `tkinter`: 직관적인 UI 화면 구성 (Python 내장 라이브러리)
  - `threading`: UI 프리징(응답 없음) 현상 방지를 위한 백그라운드 처리
  - `datetime`, `os`, `time` 등 내장 모듈

## 🚀 시작하기 (How to Use)

### 1. 사전 준비 (Prerequisites)
- 사용자의 Windows PC에 **Microsoft Outlook** 및 **PowerPoint**가 설치되어 있고, 아웃룩 계정이 로그인되어 있어야 합니다.
- Python 환경이 설치되어 있어야 합니다. (버전 3.8 이상 권장)
